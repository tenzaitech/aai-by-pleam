🔧 Troubleshooting ▼

❌ Common Issues

  • Chrome not starting: Check Chrome installation
  • Ansible error: Use WSL instead of Windows native
  • Database connection: Verify Supabase credentials
  • GPU not detected: Install CUDA drivers

✅ Solutions

# Fix Chrome issues
python debug_chrome_launcher.py

🚀 การติดตั้งและใช้งาน

📋 ความต้องการระบบ

  • Windows 10/11, macOS 10.15+, Ubuntu 18.04+
  • Python 3.8+ และ pip package manager
  • WSL (สำหรับ Ansible บน Windows)
  • Chrome Browser

🔧 ขั้นตอนการติดตั้ง

  1. Clone Repository:
    git clone <repository-url>
    cd wawagot.ai
  2. ติดตั้ง Dependencies:
    pip install -r requirements_v2.txt
  3. ตั้งค่า Environment Variables:
    # สร้างไฟล์ .env และกำหนดค่า
    OPENAI_API_KEY=your_key_here
    SUPABASE_URL=your_url_here
    SUPABASE_KEY=your_key_here
  4. ติดตั้ง WSL + Ansible (Windows):
    wsl --install
    wsl -d Ubuntu -e bash -c "sudo apt update && sudo apt install ansible"
  5. รันระบบ:
    python launch_v2.py

❌ API Key ไม่ถูกต้อง

วิธีแก้: ตรวจสอบไฟล์ .env และ config files
# ตรวจสอบ environment variables
python -c "import os; print(os.getenv('OPENAI_API_KEY'))"
